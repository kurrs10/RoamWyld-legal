Gmail Access

If you choose to use Gmail Import, you authorize Roam Wyld to request read-only access to your Gmail account via Google OAuth to search for travel confirmation emails. Specifically:

  • We scan subject lines and senders of up to 300 emails matching travel-related keywords.
  • For the emails most likely to contain booking confirmations, we retrieve up to 1,500 characters of the email body text to extract booking details (flight numbers, hotel names, confirmation codes, dates).
  • This body text is sent to Anthropic’s Claude API via our secure backend for extraction and is not stored by Roam Wyld after processing.
  • We do not access emails unrelated to travel bookings.
  • You can revoke Gmail access at any time at myaccount.google.com/permissions. Revoking access does not delete bookings already imported.

Roam Wyld’s use of Google API data complies with the Google API Services User Data Policy, including Limited Use requirements.
